Desenvolvimento de uma ferramenta para auxiliar no processo de criação da arquitetura em linha de produtos de software

Técnicas de reuso de software estão sendo cada vez mais utilizadas pelas empresas no processo de desenvolvimento de novos produtos. A Linha de Produto de Software é considerada uma das principais técnicas de reuso de software, no entanto a sua abordagem acaba sendo mais complexa que as convencionais, principalmente no processo de criação e manutenção da sua arquitetura. O mal gerenciamento de uma arquitetura pode impactar diretamente na qualidade dos produtos da organização. Tendo em vista estes aspectos o trabalho apresenta uma análise comparativa entre as principais ferramentas utilizadas para a criação e modelagem de Linhas de Produto de Software, além de propor o desenvolvimento de uma ferramenta para auxiliar no processo de arquitetura de Linha de produto. Como resultado ficou claro que o interesse por pesquisas relacionadas com Linha de Produtos de Software tem aumentado nos últimos anos, porém ainda existe uma carência de ferramentas que auxiliem no processo de arquitetura de Linha de Produto.
