Desenvolvimento de uma ferramenta para auxiliar no processo de criação da arquitetura em linha de produtos de software

Abstract: Embora a arquitetura de software não seja explicitamente definida e documentada em todos os projetos de software, a organização do sistema como um todo sempre existirá. A arquitetura também se torna crucial no contexto de Linha de Produtos de Software (SPL), devido a sua complexidade inerente. Desta forma, este trabalho tem como objetivo desenvolver uma ferramenta para auxiliar no processo de criação e documentação de arquitetura de software no contexto de SPL.
